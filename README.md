# CrosshairZ v2
**A customizable universal crosshair overlay based on QT5.**

## Installation:
1. [Download Python 3](https://python.org) and install
2. Download CrosshairZ v2 from [the main branch](https://github.com/kryvexx/CrosshairZ/archive/refs/heads/main.zip)
3. Extract the archive to a folder
4. Install requirements with `pip install -r requirements.txt`
5. Run the program with `py .\CrosshairZ.py` or `python3 .\CrosshairZ.py`
